# Interview Preparation Plan

## Fastest Preparation Order
1. Hash map/set + Two Pointers/Sliding Window
2. Stack/Queue + Binary Search
3. Intervals + Heap (Top-K)
4. Linked List + Trees/Graphs (BFS/DFS)
5. Dynamic Programming basics

---

## 30-Day Practice Plan (Day-by-Day Targets)

### Week 1: Hashing + Two Pointers
- Day 1: `Two Sum`, `Contains Duplicate`, `Valid Anagram`
- Day 2: `Group Anagrams`, `First Unique Character`, `Top K Frequent Elements`
- Day 3: `Longest Substring Without Repeating Characters`, `Valid Palindrome`
- Day 4: `3Sum` (focus on sort + two pointers), `Two Sum II`
- Day 5: `Minimum Window Substring` (hard day, go slow)
- Day 6: Review Day 1-5 mistakes, rewrite 2 problems from memory
- Day 7: Timed set (3 problems in 75 minutes)

### Week 2: Stack/Queue + Binary Search
- Day 8: `Valid Parentheses`, `Min Stack`, `Implement Queue using Stacks`
- Day 9: `Daily Temperatures`, `Next Greater Element`
- Day 10: `Binary Search`, `Search Insert Position`, `First/Last Position`
- Day 11: `Search in Rotated Sorted Array`, `Find Peak Element`
- Day 12: `Koko Eating Bananas` (binary search on answer)
- Day 13: Review weak stack and binary search problems
- Day 14: Timed set (4 problems in 90 minutes)

### Week 3: Intervals + Heap + Linked List
- Day 15: `Merge Intervals`, `Insert Interval`, `Non-overlapping Intervals`
- Day 16: `Meeting Rooms`, `Meeting Rooms II` (interval + heap)
- Day 17: `Kth Largest Element in an Array`, `Top K Frequent`
- Day 18: `Merge k Sorted Lists` (heap approach)
- Day 19: `Reverse Linked List`, `Middle of Linked List`, `Linked List Cycle`
- Day 20: `Merge Two Sorted Lists`, `Remove Nth Node From End`
- Day 21: Timed mixed set (4 problems)

### Week 4: Trees/Graphs + DP Basics
- Day 22: Tree traversal (`Preorder`, `Inorder`, `Level Order`)
- Day 23: `Maximum Depth of Binary Tree`, `Same Tree`, `Invert Binary Tree`
- Day 24: `Lowest Common Ancestor`, `Validate Binary Search Tree`
- Day 25: Graph basics: `Number of Islands`, `Clone Graph`
- Day 26: DP intro: `Climbing Stairs`, `House Robber`
- Day 27: DP practice: `Coin Change`, `Longest Increasing Subsequence`
- Day 28: Timed set (tree + graph + DP)

### Final 2 Days: Interview Simulation
- Day 29: Mock interview #1
  - 45 minutes coding + 15 minutes complexity explanation
  - Focus on communication: assumptions, edge cases, tradeoffs
- Day 30: Mock interview #2 + final revision
  - Solve 2 medium problems without notes
  - Revisit your top 10 mistakes list

---

## Daily Execution Rule (Important)
- 60-90 minutes coding
- 20 minutes review
- For every problem: write brute force first, then optimized version, then Big-O
- Keep a `mistake log` (bug type, why it happened, how to prevent)
